Overview:
Simple Quiz App is an interactive Android application developed as a university assignment. The primary objective of this coursework project is to build a straightforward quiz interface consisting of exactly 5 "Yes or No" questions to test fundamental Android development concepts.

Features:
- Interactive Quiz Flow: Navigate sequentially through questions using the Next button.
- Yes/No Selection: Simple and intuitive radio button inputs for answering.
- Show Answer Capability: A dedicated Show Answer button that navigates to a separate screen to reveal the correct answer for the current question, complete with a Back button to return to the quiz.
- Dynamic Scoring: The app tracks correct and incorrect answers. Upon completing the questions, clicking Submit calculates and displays the final score (e.g., 3/5 or 5/5) via a toast/snackbar message.
- Quiz Reset: Submitting the quiz automatically resets the application, returning the user to the first question to start over.
